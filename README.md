# QuickShaw

QuickShaw is a web application designed to address mobility challenges for people in Varanasi, especially within the IIT BHU campus. The platform allows users to view their completed rides, raise complaints, and manage their ride history with ease.

## Features

- View a list of your completed rides
- Raise complaints about specific rides
- User-friendly and responsive interface

## Things to Keep in Mind

- This project uses React and Tailwind CSS for the frontend.
- Ensure you have Node.js and npm installed on your system.

## Running in Development Mode

1. **Install dependencies:**

   ```
   npm install
   ```

2. **Start the development server:**

   ```
   npm run dev
   ```

3. **Open your browser and navigate to:**
   ```
   http://localhost:5173
   ```

You should now see the QuickShaw application running locally. Any changes you make to the source code will automatically reload the page.
